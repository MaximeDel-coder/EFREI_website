À Propos du Projet

Ce site web a été réalisé dans le cadre du cours de Programmation Web (TI504N) de la promotion INGE1-New de l'EFREI Paris Panthéon-Assas.

L'objectif est de concevoir un site vitrine pour le département informatique, en respectant les standards du web (HTML5, CSS3, JS) et en s'interdisant l'usage de frameworks lourds pour la structure principale (bien que TailwindCSS soit utilisé ici pour le prototypage rapide de la maquette).
